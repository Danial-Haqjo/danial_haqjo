دانیال حق‌جو (Danial Haqjo)

👨‍💻 برنامه‌نویس اندروید و iOS • فلاتر • جاوا • بک‌اند • اسپرینگ بوت • توسعه‌دهنده ربات‌های تلگرامی

درباره‌ من

من دانیال حق‌جو هستم؛ برنامه‌نویس چند‌سکویی با تجربه در توسعه اپلیکیشن‌های موبایل و بک‌اند. تخصص اصلی من در فریم‌ورک‌های اندروید، iOS، فلاتر و Java Spring Boot است و همچنین ربات‌های پیشرفته تلگرامی طراحی می‌کنم.

تخصص‌ها

  • توسعه اپلیکیشن‌های Android و iOS
  • برنامه‌نویسی با Java و فریم‌ورک Spring Boot
  • طراحی و توسعه Backend APIs
  • توسعه اپلیکیشن‌های چندسکویی با Flutter
  • طراحی و پیاده‌سازی ربات‌های تلگرامی
